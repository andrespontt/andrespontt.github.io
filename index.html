Hi welcome to the clipboard tool. While I am building my github io page I figured I could leave a useful tool for you to use.
This tool is a simple clipboard tool that allows you to copy text to your clipboard. You can also generate a URL with the text you have copied. You can also encode text to base64.
This is a static page and lives in your computer. It does not store any data. But I wouldn't use it to copy sensitive information since http in/out logs could be seen by someone (not me :) )
